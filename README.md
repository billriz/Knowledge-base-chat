# FieldFix AI

A mobile-first field service troubleshooting assistant for ATM and banking equipment technicians.

## Core Goals

1. Ask troubleshooting questions by symptom, machine type, or error code.
2. Capture real-world field fixes and make them searchable.
3. Build a living knowledge base from technician experience.

## MVP Features

- Mobile technician dashboard
- Ask by error code
- Ask by symptom
- Submit a fix
- Equipment lookup
- Admin review queue
- Approved fixes knowledge base

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- OpenAI
- Vercel
